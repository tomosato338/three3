「横浜迎賓館」様

サイトはこちら

概要

横浜の結婚式場の架空ホームページサイト(2作品目)

サイトの目的

①サイトの認知 ②来館予約の申し込み、資料請求、お問い合わせフォームの誘導

ターゲット

性別 女性
年齢 20～30代
悩み 予算250万以内でドレス・料理・立地の質が良い場所を探している。
式場のスタッフとの相性を大切にしている。
ペルソナについて▼

コーディングについて

素材を多く使用したため、解像度や色調の統一感を意識した。また、レスポンシブ対応においてはユーザーが見やすい画像の配置を心がけてコーディングした。

デザインについて

ターゲットのニーズに沿うように、ファーストビューで横浜の結婚式場とわかる写真を配置。jQueryの動きを上下左右に行うことで、スクロール時のユーザーのストレスを軽減させた。お問い合わせフォームは、ボタンにロゴを付け加えて強調させることでコンバージョン率の向上を図った。

使用ツール

XD / Photoshop / HTML / CSS / jQuery
